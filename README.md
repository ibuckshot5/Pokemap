# PokemapGO
### A tool to map all Pokemons around you!

Install
======
```
git clone https://github.com/ibuckshot5/Pokemap.git
cd Pokemap
npm install
```
Configure your files, then use:
```
npm start
```
to start the map.
Features
======
* Fast, hash-efficient
* Features Discord and Twitter notifications
* Easily customizable
* Supports L30+ encounters
